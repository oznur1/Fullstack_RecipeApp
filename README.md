
# A simple fullstack recipe app where users can create and view recipes. It is built using React for the frontend, Node.js and Express for the backend, and MongoDB as the database.

# Backend Framework
- express.js
- nodemon
- cors

# Frontend Framework
- react
- react-router-dom
- axios
- @tanstack/react-query
- react-select
- react-icons
- react-toastify
- tailwind
